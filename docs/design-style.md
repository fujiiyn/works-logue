# デザインスタイルガイド

Figma UIデザイン生成時のスタイルガイド。

「AIで作りました感」を排除しつつ、Works Logueの植物テーマを表現する。ムードボードで方向性確定済み。

## カラーパレット

Figma design-system (nodeId: 95:34) が正（実装時はこちらを参照）。

- 基調: ティール/セージグリーン + ウォームアイボリー背景 + ダークグリーンのテキスト
- 色は最小限。「グリーン + モノトーン」の組み合わせ（純粋なグレーモノトーンではない）
- Primary: `#29736B`（ボタン・CTA）
- Primary Dark: `#1F3833`（見出し・本文テキスト）
- Accent Teal: `#00B4CC`（ロゴ蓮の花・ブランドアクセント）
- Background: `#F7F5ED`（ページ背景）、`#FBF9F5`（カード背景）
- Border: `#E5E3DB`

## アイコン

- 絵文字は使わない
- ラインアイコン（tabler / Lucide 系）を使用

## 雰囲気

- 植物的・ボタニカル・オーガニックな印象を積極的に出す
- 参考: セージグリーン、モス、ジュニパー等の自然色パレット
